# Rust User Structure

Этот проект содержит определение структуры `User` на языке Rust с полями:
- `name`: строка с именем пользователя
- `age`: число с возрастом пользователя

## Структура файлов:
- `user.rs`: Основной файл с определением структуры и примером её использования
- `Cargo.toml`: Файл конфигурации для проекта Rust (опционально)

## Описание

Файл `user.rs` содержит:
1. Определение структуры `User` с полями `name` и `age`
2. Пример создания экземпляра структуры
3. Код для сохранения данных пользователя в файл со случайным именем

Для генерации случайного имени файла используется текущее время в наносекундах с эпохи UNIX, что дает уникальное имя для каждого запуска программы.

## Компиляция и запуск

Для компиляции и запуска программы, если Rust установлен в системе:

```bash
rustc user.rs
./user
```

После выполнения программы будет создан файл с именем в формате `user_data_<число>.txt`, содержащий имя и возраст пользователя.
