# Сервис обработки заказов (Thrift)

Этот проект реализует Thrift сервис для обработки заказов. Он включает в себя определение интерфейса сервиса, сервер и клиент для тестирования.

## Файлы проекта

- `order_service.thrift` - Определение Thrift сервиса
- `gen-py/` - Сгенерированный код Python
- `server.py` - Реализация сервера
- `client.py` - Тестовый клиент

## Структура сервиса

### Структуры данных
- `Order` - Основная структура заказа с полями: ID, имя клиента, товары, общая сумма, статус
- `GetOrderRequest` - Запрос для получения заказа по ID
- `CreateOrderRequest` - Запрос для создания нового заказа
- `OrderNotFoundException` - Исключение, выбрасываемое при попытке получить несуществующий заказ

### Методы сервиса
1. `getOrder(GetOrderRequest request)` - Получить заказ по ID
2. `createOrder(CreateOrderRequest request)` - Создать новый заказ
3. `updateOrderStatus(i32 order_id, string status)` - Обновить статус заказа

## Установка и запуск

1. Установите зависимости:
   ```
   pip install thrift
   ```

2. Сгенерируйте код из определения Thrift:
   ```
   thrift --gen py order_service.thrift
   ```

3. Запустите сервер:
   ```
   python server.py
   ```

4. В отдельном терминале запустите клиент для тестирования:
   ```
   python client.py
   ```

## Тестирование

Клиент выполняет следующие действия:
1. Создает новый заказ
2. Получает созданный заказ по ID
3. Обновляет статус заказа
4. Проверяет обновленный заказ
5. Пытается получить несуществующий заказ для проверки обработки исключений
