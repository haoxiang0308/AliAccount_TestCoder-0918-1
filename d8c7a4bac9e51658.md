## JavaScript

```javascript
function calcularFatorial(n) {
    if (n === 0 || n === 1) {
        return 1;
    }
    return n * calcularFatorial(n - 1);
}

// Exemplo de uso
const resultado = calcularFatorial(5);
console.log(`Fatorial de 5 é: ${resultado}`);
```

## SQL

```sql
SELECT 
    u.nome,
    u.email,
    p.titulo AS produto_nome,
    p.preco
FROM usuarios u
INNER JOIN pedidos p ON u.id = p.usuario_id
WHERE p.data >= '2024-01-01'
ORDER BY p.preco DESC;
```
