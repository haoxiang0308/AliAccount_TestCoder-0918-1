# PowerShell Date/Time Script

This directory contains a PowerShell script that outputs the current date and time.

## Files
- `zW7jZtps11.ps1` - The PowerShell script that displays the current date and time

## Usage
To run this script, you need to have PowerShell installed on your system.

On Windows:
```powershell
.\zW7jZtps11.ps1
```

On Linux/Mac with PowerShell Core:
```bash
pwsh zW7jZtps11.ps1
```

## Script Content
The script uses the `Get-Date` cmdlet to display the current date and time in various formats:
1. Default format
2. Custom format: yyyy-MM-dd HH:mm:ss
3. Custom format: dddd, MMMM dd, yyyy

Note: If you're running this in an environment without PowerShell, you won't be able to execute the script, but you can still view its contents.
