## Приложение для обработки чисел

Это консольное приложение на F# обрабатывает список чисел следующим образом:
1. Фильтрует, оставляя только положительные числа.
2. Возводит каждое число в квадрат.
3. Сортирует результат по убыванию.

После обработки результат сохраняется в файл со случайным именем в формате `.txt`.

### Как запустить (если .NET SDK установлен)

```bash
cd /workspace/numbers_processor_app
dotnet run
```
